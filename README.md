## Features

### Live Streaming
- **Icecast & Shoutcast Support** - Connect to any Icecast (v2.4+) or Shoutcast v2 server
- **MP3 Encoding** - Real-time encoding at 64, 96, 128, 192, 256, or 320 kbps
- **Multiple Sample Rates** - Support for 44.1kHz and 48kHz audio
- **One-Click Connect** - Save your server settings and connect instantly

### Recording
- **Video Recording** - Record broadcasts as WebM video files with your station logo overlay
- **Audio-Only Recording** - Record MP3 audio files without video
- **Automatic Segmentation** - Split recordings into 30 or 60 minute segments automatically
- **Custom Output Folder** - Choose where your recordings are saved

### Audio Quality
- **Real-Time Audio Meters** - Stereo level monitoring with peak indicators
- **Multiple Input Sources** - Select from any connected microphone or audio interface
- **Broadcast-Quality Audio** - Clean, unprocessed audio without echo cancellation or noise suppression

### Professional Interface
- **Dark Theme** - Professional, easy-on-the-eyes interface designed for studio use
- **Compact Design** - All controls accessible without scrolling
- **Status Indicators** - Clear visual feedback for streaming and recording states

## Getting Started

### 1. Configure Streaming Settings
Click the **Settings** button and enter your streaming server details:

| Setting | Description |
|---------|-------------|
| Server Type | Icecast or Shoutcast |
| Host | Your streaming server address |
| Port | Server port |
| Mount Point | Stream mount point (Icecast only) |
| Username | Server username (Icecast: usually "source") |
| Password | Your server password |
| Bitrate | Audio quality (64-320 kbps) |
| Sample Rate | 44100 Hz or 48000 Hz |

### 2. Configure Recording Settings
- Set your **Output Folder** for saved recordings
- Upload your **Station Logo** 
- Choose **Segment Duration**

### 3. Start Broadcasting
- Select your **Audio Source** from the dropdown
- Click **Start Streaming** to go live
- Click **Start Recording** to archive your broadcast
- Monitor audio levels with the real-time stereo meters

## Troubleshooting

### "Audio access denied" error
- Check Windows Privacy Settings: **Settings > Privacy > Microphone**
- Ensure antivirus software is not blocking microphone access
- Try running the app as Administrator

### "Cannot connect to server" error
- Verify your server address, port, and credentials
- Confirm your streaming server is running and accessible
- Check that your firewall allows outbound connections

### Recording not starting
- Verify your license is active and valid
- Ensure sufficient disk space is available
- Confirm the output folder exists and is writable
